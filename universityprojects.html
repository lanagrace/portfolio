Fireflies

Assignment:Hieros Gamos; Module: Creative Coding

The task was to write a generative and repetitive code which would resemble the organic, chaotic and irregular form of nature. I decided to make a night sky full of fireflies which was inspired by Van Gogh's Starry Night

DEEJI

Module: Digital Making

DEEJI is a jukebox which plays music depending on your mood. All you need to do is press an emoji that represents your mood and a Spotify playlist will start playing. This was created using electric paint and Makey Makey in a pair project. More about it here: https://natashandlanadmd.wordpress.com/blog-feed/
